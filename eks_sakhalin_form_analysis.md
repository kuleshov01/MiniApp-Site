# Анализ формы подключения карты на сайте eks.sakhalin.gov.ru

## Обзор

Форма подключения карты на сайте "Единая карта сахалинца" отправляет данные через AJAX-запрос POST на эндпоинт `/sendmail-bank-applying/`.

## Детали запроса

### URL и метод
- **URL**: `https://eks.sakhalin.gov.ru/sendmail-bank-applying/`
- **Метод**: `POST`
- **Content-Type**: `application/x-www-form-urlencoded; charset=UTF-8`


### Тело запроса (URL-encoded)

```
bank=%C2%AB%D0%93%D0%B0%D0%B7%D0%BF%D1%80%D0%BE%D0%BC%D0%B1%D0%B0%D0%BD%D0%BA%C2%BB&bank-id=gazp&forwhom=%D0%9D%D0%B0+%D1%81%D0%B5%D0%B1%D1%8F&forwhom-id=self&lastname=%D1%82%D0%B5%D1%81%D1%82&name=%D1%82%D0%B5%D1%81%D1%82&patron=%D1%82%D0%B5%D1%81%D1%82&mobPhone=%2B7+(999)+999+99+99&rstkAgree=on&dolgoletieAgree=on&aquaAgree=on&persAgree=on&n=1769614451181
```

### Декодированные поля формы

| Поле | Значение | Обязательно | Описание |
|------|----------|-------------|----------|
| `bank` | «Газпромбанк» | Да | Название выбранного банка |
| `bank-id` | `gazp` | Да | Идентификатор банка |
| `forwhom` | На себя | Да | Тип заявки (для кого карта) |
| `forwhom-id` | `self` | Да | Идентификатор типа заявки |
| `lastname` | тест | Да | Фамилия заявителя |
| `name` | тест | Да | Имя заявителя |
| `patron` | тест | Нет | Отчество заявителя |
| `mobPhone` | +7 (999) 999 99 99 | Да | Мобильный телефон |
| `rstkAgree` | `on` | Нет | Согласие на использование на транспорте |
| `dolgoletieAgree` | `on` | Нет | Согласие на использование в "Сахалинском долголетии" |
| `aquaAgree` | `on` | Нет | Согласие на использование в Аква Парке (дети) |
| `persAgree` | `on` | Да | Согласие на обработку персональных данных |
| `n` | `1769614451181` | Да | Timestamp/номер заявки |

### Поля формы на сайте

1. **Банк-партнер** * → `bank`, `bank-id`
2. **На кого оформляется карта** * → `forwhom`, `forwhom-id`
3. **Фамилия** * → `lastname`
4. **Имя** * → `name`
5. **Отчество** → `patron`
6. **Мобильный телефон** * → `mobPhone` (формат: +7(___)___-__-__)
7. **Я хочу использовать "Единую карту сахалинца" на транспорте** → `rstkAgree`
8. **Я хочу использовать "Единую карту сахалинца" в "Сахалинском долголетии"** → `dolgoletieAgree`
9. **Я хочу использовать "Единую карту сахалинца" в Аква Парке (только для детей)** → `aquaAgree`
10. **Я согласен на обработку персональных данных** * → `persAgree`
